二、form标签（post请求）

解释：
1、form表单里面的input的type没有submit属性，就无法提交；
2、label标签的for=yyy 与 input标签的id=yyy 同时出现，点击文字就能聚焦到输入框，或用label套input也行；

get默认会把参数出现在查询参数上；
post会把参数放入请求的第四部分；
我们不能让get请求出现在第四部分；
post可以把参数出现在查询参数，在action=”users/use?name=1”;
